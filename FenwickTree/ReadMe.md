## Problem Statement

You are given an array a0, a1,....,an-1 of length N. Process Q queries of the following types
* '0 p x': ap <-- ap + x
* '1 l r': print sum from l to r-1 inclusive

### Sample Input

5 5

1 2 3 4 5

1 0 5

1 2 4

0 3 10

1 0 5

1 0 3


### Sample Output
15

7

25

6
